## Resumen Proyecto

Piñata Vision es una función de Viva Piñata: Trouble in Paradise.

Puedes escanear una carta Piñata Vision con la cámara **Xbox LIVE Vision** para importar su objeto al juego.

Agradecimiento especial a Peter Jensen por crear el PV Creator original para iOS y compartir el repositorio [pv-decoder](https://github.com/pinatavision/pv-decoder).

<br/>

## 📦 Uso

Script decodificador para identificar datos codificados en un código de barras de Piñata Vision.

```bash
python pv_decoder.py <archivo_entrada> [-d]
```

Script codificador para generar códigos de barras de Piñata Vision a partir de los datos decodificados.

```bash
python pv_encoder.py <archivo_entrada> [-d]
```

## 🃏 Cartas Personalizadas

[PV Creator](https://play.google.com/store/apps/details?id=com.raks.pvcreator&hl=es) es una aplicación Android que simplifica la creación de cartas personalizadas de Piñata Vision. Todas las cartas generadas con esta aplicación son de uso gratuito e inmediatamente reutilizables.
